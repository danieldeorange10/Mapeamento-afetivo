Sobre o Projeto de Mapeamento Interativo

O projeto de Mapeamento Interativo é uma aplicação web desenvolvida para permitir a visualização e interação com marcadores que representam escolas e praças em uma determinada região.

Recursos Principais

  • Visualização de marcadores no mapa
  • Informações detalhadas sobre cada marcador, incluindo nome, imagem e detalhes do local
  • Possibilidade de adicionar novos marcadores
  • Integração com a API do YouTube para exibir vídeos relacionados aos locais

Tecnologias Utilizadas

  • Leaflet - biblioteca JavaScript para exibição de mapas interativos
  • Node.js - plataforma de desenvolvimento backend
  • MongoDB - banco de dados NoSQL para armazenamento de dados
  • HTML, CSS e JavaScript - linguagens para desenvolvimento web

Equipe de Desenvolvimento

O projeto foi desenvolvido por uma equipe de programadores e designers especializados em desenvolvimento web e experiência do usuário.
